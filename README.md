## Installation  
1. Clone the repository:  
   ```sh
   git clone https://github.com/rohanbindelowar/ai-image-recognition.git
   cd ai-image-recognition
   ```
2. Install dependencies:  
   ```sh
   npm install
   ```
3. Run the app:  
   ```sh
   npm start
   ```
